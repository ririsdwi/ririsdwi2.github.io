Project Blender

Project 1

Project 3D ini merupakan hasil eksplorasi saya di aplikasi Blender, terinspirasi dari gaya Minicraft. Proses pembuatannya cukup menantang karena membutuhkan ketelitian dalam membangun detail objek dan tekstur agar tampak menarik dan presisi.

Project 2

Project cermin ini menantang karena membutuhkan ketelitian dalam membuat refleksi dan detail permukaan. Hasil akhirnya memberikan kesan realistis dan memuaskan secara visual (CERMIN).

Project 3

Project asbak rokok ini menantang karena membutuhkan ketelitian tinggi dalam membuat detail bentuk dan tekstur. Hasil akhirnya tampak realistis dan memuaskan secara visual (ASBAK ROKOK).

Project 4

Project kaleng susu ini menonjolkan desain sederhana namun elegan, dengan detail bentuk dan tekstur yang dibuat agar tampak realistis dan menarik secara visual (KALENG SUSU).

Project 5

Project ini adalah model 3D kran air yang dibuat untuk kebutuhan sehari-hari. Desainnya sederhana namun fungsional, menonjolkan detail bentuk dan tekstur agar tampak realistis dan mudah dikenali (KRAN AIR).

Project 6

Project kursi ini merupakan hasil eksplorasi saya dalam membuat model furnitur 3D di Blender. Proses pembuatannya melatih ketelitian dalam membangun bentuk dan proporsi agar tampak realistis dan nyaman dipandang.

Project 7

Project jam dinding ini menampilkan desain minimalis namun tetap elegan, dengan detail bentuk dan tekstur yang dibuat agar tampak modern dan menarik secara visual (JAM DINDING).

Project 8

Project ini adalah sebuah buku dengan cover yang saya desain sendiri, menggabungkan kreativitas dan teknik desain grafis untuk menghasilkan tampilan yang unik dan menarik. Setiap detail pada cover dibuat dengan penuh perhatian agar buku ini tampil menonjol dan profesional.

Project 9

Project ini merupakan salah satu karya 3D favorit saya, menantang untuk dikerjakan namun sangat memuaskan saat melihat hasil akhir animasi meja dan taplak yang detail dan realistis.

Project 10

Project ini merupakan model 3D gelas kaca yang membutuhkan ketelitian tinggi dalam pembuatan detail transparansi dan refleksi. Prosesnya menantang, namun hasil akhirnya tampak realistis dan memuaskan secara visual (GLASS).

Project 11

Project Hendril ini merupakan salah satu karya 3D yang paling menantang, karena membutuhkan ketelitian tinggi dalam pembuatan detail bentuk dan tekstur. Proses pengerjaannya melatih kesabaran dan kreativitas untuk menghasilkan hasil akhir yang realistis dan memuaskan.

Project 12

Project ini adalah animasi 3D vas bunga yang menampilkan detail bentuk dan tekstur bunga serta vas secara realistis. Proses pembuatannya melibatkan teknik modeling, texturing, dan animasi untuk menciptakan tampilan yang hidup dan menarik (VAS BUNGA).
